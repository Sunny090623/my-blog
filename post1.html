从零部署毛玻璃博客

2025-02-18 6分钟前 教程 · Cloudflare · 前端

最近好多朋友问我这种毛玻璃效果的博客是怎么做的，今天就出一篇保姆级教程，从零开始部署一个属于自己的粉色毛玻璃博客，并且免费托管在 Cloudflare Pages 上。

1. 编写基础 HTML

首先创建一个 index.html，引入 Font Awesome 图标库，然后构建基本的页面结构：导航栏、个人资料卡、文章卡片网格、页脚。

关键在于使用 backdrop-filter: blur(12px) 实现毛玻璃效果，配合半透明背景 rgba(255,255,255,0.2)，让背景透出来。

2. 添加哔哩哔哩主题色

主色调 #FB7299 用在图标、标题、悬停边框上。可以通过渐变背景、阴影等手法让颜色更丰富。

3. 部署到 Cloudflare Pages

将代码推送到 GitHub 仓库，在 Cloudflare Pages 中连接仓库，设置构建输出目录为根目录，即可自动部署。还可以绑定自定义域名，免费开启 HTTPS。

更多细节可以看我的其他文章，或者直接查看 Cloudflare Pages 官方文档。
